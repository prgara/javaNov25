## Linux Commands

- touch -> creating a new file
- ls -> List the files -> Show me what's here
- whoami -> tells the name of the machine/user
- vim -> write to a file or editor
- clear -> clear the screen

editor -> vim, nano, emacs 
I -> Insert mode activates
Esc -> Exit from insert mode
:wq -> save & exit
:w -> Save only
:q -> quit
:q! -> quit w/o saving
:x -> save & exit

Folders -> Directory
pwd -> Show to directory -> present working directory
mkdir -> create a new folder
cd -> change directory
cd .. -> one step back
cd ~ -> home directory
cp -> copy content from one file to another
mv -> rename file
rm -> remove a file
rmdir -> remove a folder
rm -r -> remove a folder with content inside
ls -l -> list directory with info
